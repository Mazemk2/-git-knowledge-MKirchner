# Git Wissensspeicher

## Git Befehle

### git init
Initialisiert ein neues Git-Repository.

### git status
Zeigt den aktuellen Status des Repositories an.

### git add
Fügt Dateien zur Staging-Area hinzu.

### git commit
Erstellt einen neuen Commit mit den aktuellen Änderungen.
